# Layout

## Aspect ratio

Enhance your web layout with the `aspect-ratio` CSS property! By defining a preferred aspect ratio for your box, this powerful feature facilitates seamless auto-sizing and layout functions. Unlock new possibilities for responsive design and achieve optimal visual balance effortlessly.

<div className='table_container'>
| `Classes    ` | `Properties         ` |
| ------------- | --------------------- |
| aspect-auto   | aspect-ratio : auto;  |
| aspect-auto   | aspect-ratio : 1;     |
| aspect-auto   | aspect-ratio : 16/9;  |
| aspect-0      | aspect-ratio : 1/2;   |

</div>
**Usage :**

To leverage the Aspect Ratio utility classes, adhere to the following pattern:

{ aspect }-{ auto | square | 16 | 0 }

Example:

```jsx
<div class="bg-akram w-100-100">
<img class="aspect-auto" />
<img class="aspect-square" />
<img class="aspect-16" />
<img class="aspect-0" />
</div>
```

By employing these classes, you can easily control the aspect ratio of elements within the container, achieving responsive and visually appealing designs.

## Float

The `float ` CSS property positions an element on the left or right side of its container, enabling text and inline elements to wrap around it. This takes the element out of the normal flow of the page, while still being a part of the flow (unlike absolute positioning).


<div className='table_container'>

| `Classes    ` | `Properties     ` |
| ------------- | ----------------- |
| float-left    | float : left ;    |
| float-right   | float : right ;   |
| float-none    | float : none ;    |

</div>
**Usage :**

To utilize Float utility classes, follow this pattern:

{float}-{left|right|none}

Example:

```jsx
<div class="bg-primary">
<div class="">
<img class="float-left" src="/static/images/akram.png" />
<p class="text-akram">AkramCSS</p>
</div>
<div class="transition-color duration-slow ease-in-out delay-fase">
<img class="float-right" src="/static/images/akram.png" />
<p class="text-akram">AkramCSS</p>
</div>

<img class="float-none" src="/static/images/akram.png" />
<p class="text-akram">AkramCSS</p>
</div>
</div>
```

## Object fit

<div className='table_container'>

| `Classes           ` | `Properties               ` |
| -------------------- | --------------------------- |
| objectF-fill         | object-fit : fill ;         |
| objectF-contain      | object-fit : contain ;      |
| objectF-cover        | object-fit : cover ;        |
| objectF-none         | object-fit : none ;         |
| objectF-scale-down   | object-fit : scale-down ;   |

</div>
**Usage :**

To utilize Object Fit utility classes, follow this pattern:

{objectF}-{fill|contain|cover|none|scale-down}

Example:

```jsx
<div class="bg-primary">
<div class="">

</div>
</div>
```

## Object position

The `object-position` CSS property defines the alignment of the selected replaced element's content within its box. If the replaced element's object doesn't fully cover the box, the element's background will be visible in the uncovered areas. Moreover, you can control how the replaced element's object's original size is adjusted to fit the box by using the object-fit property.


<div className='table_container'>

| Classes              | Properties                       |
| -------------------- | -------------------------------- |
| objectP-center       | object-position : center ;       |
| objectP-top          | object-position : top ;          |
| objectP-bottom       | object-position : bottom ;       |
| objectP-left         | object-position : left ;         |
| objectP-right        | object-position : right ;        |
| objectP-left-bottom  | object-position : left bottom ;  |
| objectP-left-top     | object-position : left top ;     |
| objectP-right-bottom | object-position : right bottom ; |
| objectP-right-top    | object-position : right top ;    |

</div>
**Usage :**

To utilize Object Position utility classes, follow this pattern:

{objectP}-{bottom|center|left|right|top|left-bottom|left-top|right-bottom|right-top}

Example:

```jsx
<div class="bg-primary">
<div class="">
<img class="objectF-none objectP-center" src="/static/images/akram.png" />
<img class="objectF-none objectP-right" src="/static/images/akram.png" />
<img
class="objectF-none objectP-left-bottom"
src="/static/images/akram.png"
/>
<img class="objectF-none objectP-left" src="/static/images/akram.png" />
<img
class="objectF-none objectP-right-bottom"
src="/static/images/akram.png"
/>
</div>
</div>
```
