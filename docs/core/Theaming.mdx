# How to use

Discover the seamless world of theming with AkramCSS! Utilizing themes has never been easier. By default, our library seamlessly adapts to your browser's theme. However, if you wish to customize it to your liking, it's just as effortless. Simply apply the **`dark-mode`** or **`light-mode`** class to the specific element where you want to see the theme change take effect. Empower your web design with AkramCSS theming and unleash the full potential of personalized user experiences!

```jsx
<div className="display-flex flex-row ">
  <div className="bg-sissa rounded-lg p-1 ">
    <p className="text-md font-medium">I Love AkramCSS theming</p>
  </div>
  <div className="bg-sissa rounded-lg p-1 dark-mode">
    <p className="text-md font-medium">I Love AkramCSS theming</p>
  </div>
</div>
```

## Create your first Markdown Page

Create a file at `src/pages/my-markdown-page.md`:

```mdx title="src/pages/my-markdown-page.md"
# My Markdown page

This is a Markdown page
```

A new page is now available at [http://localhost:3000/my-markdown-page](http://localhost:3000/my-markdown-page).
